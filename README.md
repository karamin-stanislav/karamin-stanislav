# ksp-hub.github.io (Root GitHub Pages)

Этот набор файлов предназначен для **отдельного репозитория GitHub Pages** с именем `ksp-hub.github.io`. Репозиторий нужен для того, чтобы домен `https://ksp-hub.github.io/` (корень организации/пользователя) отдавал корректный ответ и перенаправлял посетителей на основное портфолио `https://ksp-hub.github.io/Stanislav-Karamin-Portfolio/`.

## Содержимое

- `index.html` — страница с мгновенным редиректом на портфолио и метатегами (description, canonical, Yandex verification).
- `robots.txt` — файл с инструкциями для поисковых роботов (разрешён весь сайт, указана карта сайта).
- `sitemap.xml` — карта сайта с основным URL портфолио.

## Что нужно сделать

1. Создайте новый репозиторий в GitHub с точным названием `ksp-hub.github.io`.
2. Добавьте в него эти три файла (можно просто распаковать архив в корень репозитория).
3. В настройках репозитория убедитесь, что включён GitHub Pages (по умолчанию он раздаётся из ветки `main`).
4. После публикации у URL `https://ksp-hub.github.io/` появится 200-ответ и редирект на портфолио.
5. В Яндекс.Вебмастере ресурс `https://ksp-hub.github.io/` пройдёт проверки главной страницы, `robots.txt` и `sitemap.xml`.

> Примечание: архив с этими файлами находится в каталоге `archives/` основного проекта, файл `ksp-hub-github-io-root.zip`.
